# 🔥 SentinelRecon

A powerful Python-based reconnaissance tool for cybersecurity professionals.

---

## 🚀 Features

- 🔍 WHOIS Lookup
- 🌐 DNS Enumeration
- 🛣️ Traceroute
- 🔒 Port Scanning
- 🧠 Service Detection
- 🌍 Web Server Detection
- 📡 UDP Scan
- ⚠️ Vulnerability Scan (Nmap Scripts)
